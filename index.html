What is Unix time?

Unix time is the number of seconds that have elapsed since January 1st, 1970. It is used in computing and programming as a way of storing time in a universally recognised format.


January 19, 2038

January 19, 2038 is a similar bug to Y2K. Unix time will get too large to be stored as a value in 32-bit signed integer and thus may cause many problems if the issue isnt fixed.
