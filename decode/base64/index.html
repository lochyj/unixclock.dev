Why use base64?

Base64 is an encoded binary that is stored in the same 64 characters all the time. It is used for sending data over connections that have different protocols; such as FTP. FTP translates line endings but with base64 this isn't a problem.

Encode Base64 in other languages

base64.b64encode(b'String') Javascript: btoa(String) PHP: base64_encode('String') C#: System.Convert.ToBase64String(System.Text.Encoding.UTF8.GetBytes("String")) Java: Base64.getEncoder().encodeToString("String".getBytes()) Ruby: Base64.encode64("String") C: base64_encode("String") C++: base64_encode("String") Haskell: Base64.encode64 "String" Rust: base64::encode("String") Go: base64.StdEncoding.EncodeToString("String") Swift: base64.encode("String")
