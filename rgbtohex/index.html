Convert RGB to HEX in other languages


Python: '%02x%02x%02x' % rgbValue Javascript: '#' + rgbValue.map(function(x){return ('0' + parseInt(x).toString(16)).slice(-2)}).join('') PHP: sprintf('%02x%02x%02x', rgbValue) Ruby: "%02x%02x%02x" % rgbValue Java: String.format("%02x%02x%02x", rgbValue) C#: string.Format("{0:x2}{1:x2}{2:x2}", rgbValue) C++: std::stringstream ss; ss << std::hex << rgbValue; return ss.str(); C: sprintf( "%02X%02X%02X", rgbValue );

Convert HEX to RGB in other languages


Python: [int(x, 16) for x in re.findall(r'..', hexValue)] Javascript: hexValue.substr(1).match(/.{2}/g).map(function(x){return parseInt(x, 16)}) PHP: sscanf(hexValue, "%2x%2x%2x") Ruby: hexValue.scan(/../).map{|x| x.hex} Java: Integer.parseInt(hexValue, 16) C#: int.Parse(hexValue.Substring(1), System.Globalization.NumberStyles.HexNumber) C++: std::stringstream ss; ss << std::hex << hexValue; return ss.str(); C: sscanf(hexValue, "%02x%02x%02x")
